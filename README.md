# InboxPilot — AI-ассистент для почты

![Preview](https://cdn.poehali.dev/templates/ai-email-copilot-ru/preview.jpg)

Лендинг AI-сервиса для автоматизации email-переписки. Минималистичный дизайн с интерактивными мокапами интерфейса.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- shadcn/ui
- Lucide Icons
